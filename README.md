# Zerodha-clone-stock
A full-stack MERN Zerodha clone featuring a dashboard to track holdings, positions, and stock data. Built with React for UI and Node.js/Express for APIs, with MongoDB for storage. Includes chart visualization and a basic Buy/Sell interface to simulate trading workflows.

## 🚀 Getting Started

### Prerequisites
- Node.js installed
- MongoDB database (Atlas or local)

### Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/shivratnadhoble/Zerodha-clone-stock.git
   ```

2. **Backend Setup:**
   - Go to the `backend` folder.
   - Run `npm install`.
   - Create a `.env` file based on `.env.example` and add your `MONGO_URI`.
   - Start the server: `npm start`.

3. **Frontend & Dashboard Setup:**
   - For both `frontend` and `dashboard` folders:
   - Run `npm install` inside each folder.
   - Start the development server: `npm start`.

## 🌐 Deployment

To see the website live, we recommend the following:

- **Frontend & Dashboard**: Deploy to [Vercel](https://vercel.com/) or [Netlify](https://www.netlify.com/).
- **Backend**: Deploy to [Render](https://render.com/) or [Railway](https://railway.app/).

Once deployed, you will receive a public URL to share with others.
